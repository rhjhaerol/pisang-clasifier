# pisang-clasifier
Ini proyek klasifikasi pisang berdasarkan kondisi luka dan tidak luka menggunakan arsitektur Convolutional Neural Network (CNN), ResNet50, dan VGG16. Proyek ini bertujuan untuk mengembangkan model pembelajaran mesin yang dapat membedakan pisang yang mengalami luka atau cedera dengan pisang yang dalam kondisi baik menggunakan pendekatan yang berbeda.
